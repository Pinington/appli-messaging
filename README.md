# Application chat en ligne
Projet de Techno Web II, qui consiste a faire une application de chat en ligne

## Requis
 - Django  
 - JQuery
 - BootStrapJS

## Fonctionnalitees
On feras une application de chat basique, capable d'envoyer des characteres UTF-8.  
On essayeras d'implementer une page de login pour differencier les utilisateurs.
